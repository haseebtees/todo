# Agile Documentation Journal – Weekly Agile Log

**Technology Stack:** Kotlin, Jetpack Compose, Firebase  
**Methodology:** Scrum-based Agile Development

## Week 1 – Sprint Planning & Authentication Development

### Sprint Objectives
- Finalize application requirements and UI flow
- Design authentication-related screens
- Integrate Firebase Authentication

### Work Completed
- Implemented Splash Screen with application branding
- Created Login, Signup, and Reset Password UI using Jetpack Compose
- Integrated Firebase Email/Password Authentication
- Implemented logout functionality

### Screenshots Evidence
- Splash Screen  
- Login Screen  
- Signup Screen  
- Reset Password Screen  
- Logout Screen  

### Problems Encountered
- Authentication error handling issues

- Refined Compose layouts
